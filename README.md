# Dataficação e Racismo Algorítmico na IA Generativa

Projeto web estático desenvolvido em HTML, CSS e JavaScript para apresentar o trabalho de Cultura de Poder e Relações Raciais.

## Como abrir no computador

1. Extraia o arquivo `.zip`.
2. Abra a pasta `trabalho-racismo-algoritmico` no Visual Studio Code.
3. Abra o arquivo `index.html` no navegador, ou use a extensão **Live Server**.

## Versão web e mobile

O projeto já é responsivo, então funciona em computador e celular pelo mesmo link. Também inclui `manifest.json` e `service-worker.js`, permitindo adaptação para PWA quando publicado em um servidor com HTTPS, como GitHub Pages, Netlify ou Vercel.

## Publicação

Para tornar o site acessível para outras pessoas, publique a pasta em uma hospedagem estática, como:

- GitHub Pages
- Netlify
- Vercel

Depois de publicado, o mesmo link abrirá a versão desktop e a versão mobile automaticamente.
